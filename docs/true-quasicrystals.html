What Makes a True Quasicrystal?

Unlike the diffraction patterns we showed earlier, these generators create actual quasicrystal structures:

  • Aperiodic: Never repeats exactly (no translational symmetry)
  • Long-range order: Produces sharp diffraction peaks
  • Forbidden symmetry: 5-fold, 7-fold, or other non-crystallographic symmetries
  • Discrete or continuous: Can be point sets (atoms) or density fields

Method A: Penrose Tiling

How it works: Uses inflation/deflation rules on Robinson triangles. Each generation splits triangles according to precise rules involving the golden ratio φ = (1+√5)/2. This creates a discrete aperiodic tiling—a true 2D quasicrystal.

Method B: Wave Interference with Incommensurable Frequencies

How it works: Superimposes plane waves with carefully chosen wave vectors. Uses two sets of waves with wavelengths in the golden ratio (incommensurable). This creates a continuous density field ρ(x,y) with quasicrystalline properties.

ρ(x,y) = Σ cos(q⃗ᵢ·r⃗ + φᵢ) where |q⃗₂|/|q⃗₁| = φ (golden ratio)

How to Verify a True Quasicrystal

  1. Translation Test: Shift pattern by any vector → never matches exactly
  2. Fourier Transform: Sharp peaks (not broad) with forbidden symmetry
  3. Inflation Test: For Penrose, inflating preserves structure with φ scaling
  4. Density Analysis: For waves, check incommensurability of frequencies
